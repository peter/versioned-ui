# versioned-ui - A CMS Admin UI

## Starting the Development Server

```
yarn install
yarn serve
open http://localhost:8080
```

To create a production build, run `yarn build`

## How this App Was Created

```
npm install -g @vue/cli
vue create versioned-ui
```
